fun Group.deepShrink(shrinkParents: Boolean = false, invalidateParents: Boolean = true)
Content copied to clipboard

Reduces the whole of the element to (0, 0) and invalidates hierarchy, effectively reducing the size of this group to the minimum.

This method recursively shrinks all children of the group except for Stacks. That can cause custom WidgetGroups to break if their elements are arranged manually.

Parameters

invalidateParents

whether to invalidate hierarchy. Has no special effect if shrinkParents is true.

shrinkParents

whether to shrink all parents except the scene root. Use with caution.
